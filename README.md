# Kilogram8

Kilogram8 is the best in class messenger. 

## Pre-requisites

You should have `NodeJS` installed 

You should know `js`

## Getting started

Install a load of dependencies

`npm install` or `yarn`

Launch the app

`npm start` or `yarn start`
